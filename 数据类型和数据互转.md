## 数据互转

> Map -> Array
```
arr = Array.from(map)
```

> Map -> Set
```
set = new Set(map.values())
```

> Map -> Object
```
obj = Object.fromEntries(map.entries())
```

> Object -> Map
```
map = new Map(Object.entries(obj))
```

> Object -> Set
```
set = new Set(Object.values(obj))
```

> Object -> Array
```
Object.values(obj)
Object.keys(obj)
```

> Set -> Array
```
arr = [...set]
```
